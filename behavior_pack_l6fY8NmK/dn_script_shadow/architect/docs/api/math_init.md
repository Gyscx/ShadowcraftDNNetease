# 数学 (Math) 模块 (`__init__.py`)

`architect.math.__init__.py` 是数学系统的入口模块。

该文件当前为空，可能用于未来扩展或作为包导入的占位符。

## 相关子模块

数学系统包含以下子模块：

1. **`double.py`** - 双精度数学运算。
2. **`mat4.py`** - 4x4 矩阵运算。
3. **`unit.py`** - 单位转换和数学常量。
4. **`utils.py`** - 通用数学工具函数。
5. **`utilsServer.py`** - 服务端专用数学工具。
6. **`vec3.py`** - 三维向量运算。

## 使用建议

由于该文件为空，建议直接导入具体的子模块：


## 数学系统概述

数学系统提供了游戏开发中常用的数学运算功能，包括：

1. **向量运算**: 三维向量的加减乘除、点积、叉积、归一化等。
2. **矩阵运算**: 4x4 矩阵的创建、变换、乘法等。
3. **插值运算**: 线性插值、平滑插值等。
4. **单位转换**: 角度与弧度转换、距离单位转换等。
5. **几何计算**: 距离计算、碰撞检测、射线检测等。
